MobiBench

This website represents the online documentation for the MobiBench framework, which is used to benchmark mobile reasoners.

This website also serves as the online documentation for the paper "Optimizing and Benchmarking OWL2 RL for Semantic Reasoning on Mobile Platforms", submitted to the Journal of Web Semantics (2016). In this capacity, the website also lists large-scale version of figures 3-8 here (which were cut off due to high peaks in the paper).

All content is released under the Apache License 2.0.
